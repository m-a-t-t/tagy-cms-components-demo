Editable directive!

Just add "tagy-cms-editable" attribute to an element and you can edit its content. Click me or select from auto generated editable elements from the .

Editing will update HTML markup directly so it can be used for front-end CMS system.

Angular events are used and decoupled
components so you can extend behaviour easily. For example you could add mouse over effects and other UI feedback methods.

checkout at GitHub or use bower install angular-edit-inline
